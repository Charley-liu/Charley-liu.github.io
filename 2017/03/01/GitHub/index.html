关于Github搭建博客

日志二级目录
  • github

摘要：关于该博客的安装，调试的体会！！！！

其实我刚开始想做的时候就有点纠结！因为会配置很多的文件啊！还要安装很多的插件和程序，想想都头疼！！
对于github我是一个新手！没玩几天，到现在还没有真正明白github的大部分功能！你就凑合看吧！哈哈哈哈！！！
这个博客其实重点就在于hexo的应用上，还有配置路径上，搞懂这些其实就容易的多。
网上攻略很多，我推荐你一个，我使用的搭建github步骤，真心不错！每一步都讲的很细，你可以好好看看，对你会有帮助的！相信我！！！
对了，还有一样东西也一道给你们，就是markdown的书写格式，就是你们看到的.md文件的编写规范，很受用哦！
markdown简明语法!!!
还有一件事情，就是我更新这次的原因，如果你这次写完博客以后，下次再想继续编写，更新博客，记得要在blog的文件上右击，否则无法连接github，也更新不了。或者有可能会把你以前写的所有东西都被覆盖。所以要谨慎！！
2017年8月2号更新
对于要想在另外一台电脑上继续编写你的博客，我用的是最笨的一种方法，就是把你的blog文件夹，在你搭建的电脑上拷贝下来，然后放在你想要写东西的电脑上，在这个电脑上装上hexo以后，直接hexo init以后，在输入hexo s -g这个指令本地只要能打开，你就可以在这台电脑上继续编写你的博客了，这种只适合更新博客比较慢的同学，因为考一次文件挺麻烦的。。。。记得下次在回到原来电脑上写，还要在拷回去。。。。。
方法有点二，如果有别的方式记得告诉我。。。。。谢过
